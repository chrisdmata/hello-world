The design for a pressure-sensitive authentication system, where the resistors (the dark circles) would be placed under a touchscreen to read in pressure values.

The database implentation. Pressure values, time values, and the actual password are shown here. The users in the database can be seen in the back window.

The touchscreen display, placed atop the force-sensitive resistors. The arduino reads time and pressure values, and sends them to the database in the raspberry pi.

The resistors themselves. The arduino is coded in C to read in values when they are greater than 0 (when any force is placed on them) and communicate with the Raspberry Pi.
